# Listas de Exercícios - Estruturas de Dados Orientadas a Objetos (CIN0135)

Este repositório contém as soluções para as listas de exercícios da disciplina de **Estruturas de Dados Orientadas a Objetos** (CIN0135) do período 24.2, implementadas em C++. Todas as questões foram avaliadas através da plataforma de juiz online [VJudge](https://vjudge.net/).

## Conteúdo das Listas

1. **Lista 1 - Listas Ligadas, Filas e Pilhas**: Implementação e manipulação de estruturas de dados sequenciais
2. **Lista 2 - Hashing e BSTs**: Tabelas de dispersão e árvores de busca binária
3. **Lista 3 - Final List EDOO**: Exercícios complementares e revisão

## Tecnologias

- **Linguagem**: C++ 23
- **Plataforma de avaliação**: VJudge
